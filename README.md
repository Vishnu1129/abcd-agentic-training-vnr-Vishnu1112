# abcd-agentic-training-vnr-Vishnu1112

# 🤖 AI Chatbot with Resume Context & Evaluation

## 📌 Overview

This project is an **AI-powered chatbot** built using Large Language Models (LLMs) that can answer questions based on a user's **resume/profile data**.

It simulates real-world interactions such as:

* Responding to recruiters
* Answering interview questions
* Explaining skills and experience

The chatbot is integrated with an **evaluation system** to assess response quality.

---

## 🚀 Features

* 💬 Conversational AI chatbot
* 📄 Resume-based context understanding
* 🧠 Uses LLMs via Ollama
* 📊 Response evaluation system
* 🌐 Interactive UI using Gradio

---

## 🛠️ Tech Stack

* **Python**
* **Ollama (LLM)**
* **Gradio (UI)**

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/Vishnu1129/abcd-agentic-training-vnr-Vishnu1112.git
cd abcd-agentic-training-vnr-Vishnu1112
```

---

### 2. Install dependencies

```
pip install -r requirements.txt
```

---

### 3. Run Ollama

Make sure Ollama is installed and running:

```
ollama run llama3
```

---

### 4. Run the notebook

Open:

```
3_lab3.ipynb
```

Run all cells to start the chatbot.
